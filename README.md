# Vue.js Portfolio Project

A modern Vue.js web application built with Vite and Vue Router, displaying a Hello World homepage.

## Project Structure

```
src/
├── router/
│   └── index.js           # Vue Router configuration
├── views/
│   └── HomeView.vue       # Home page with Hello World
├── components/            # Reusable Vue components
├── assets/                # Static assets
├── App.vue                # Root component
├── main.js                # Application entry point
└── style.css              # Global styles
```

## Setup & Installation

This project uses **Yarn** as the package manager with **Vue 3** and **Vite**.

### Prerequisites
- Node.js 20.15.0 or higher
- Yarn package manager

### Install Dependencies

```bash
yarn install
```

## Development

Start the development server with hot module reloading:

```bash
yarn dev
```

The application will be available at `http://localhost:5173`

## Build

Build the project for production:

```bash
yarn build
```

## Preview

Preview the production build locally:

```bash
yarn preview
```

## Technologies Used

- **Vue 3** - Progressive JavaScript framework
- **Vite** - Next generation frontend build tool
- **Vue Router 5** - Official router for Vue.js
- **Yarn** - Package manager

## Features

- ✅ Vue Router setup with basic home view
- ✅ Hello World homepage with gradient background
